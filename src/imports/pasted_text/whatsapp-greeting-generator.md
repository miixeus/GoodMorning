Create a modern, clean, and highly simple mobile-first web app UI for a tool that generates shareable greeting images for WhatsApp.

TARGET AUDIENCE:
People aged 40+, not tech-savvy. The interface must be extremely simple, readable, and intuitive.

STYLE:

* Minimalist
* Friendly and warm
* Large buttons
* High contrast
* Rounded elements
* Soft shadows
* Accessible typography

COLOR PALETTE:

* Primary: #00cf40 (green)
* Secondary: #5f19ea (purple)
* Background: #ffffff
* Text: #000000
* Light gray for sections: #f5f5f5

TYPOGRAPHY:

* Large font sizes
* Clear hierarchy
* Sans-serif (like Inter or Roboto)

---

SCREENS TO DESIGN:

1. HOME SCREEN

Elements:

* Title: "Crie sua mensagem"

* Subtitle: "Escolha uma opção abaixo"

* 3 large buttons:

  * "🌞 Bom dia"
  * "🌇 Boa tarde"
  * "🌙 Boa noite"

* Input field:
  Placeholder: "Digite seu nome (opcional)"

* Main CTA button (very prominent):
  "Gerar imagem"

---

2. RESULT SCREEN

Elements:

* Generated image preview (large card with rounded corners)

* The image must contain:

  * Background (soft gradient or nature style)
  * Dynamic text:
    Example: "Bom dia, João! ☀️ Que seu dia seja incrível!"

* Buttons below:

  * "Compartilhar no WhatsApp" (green, primary)
  * "Gerar outra imagem" (secondary)

---

3. IMAGE TEMPLATE VARIATIONS (IMPORTANT)

Create multiple background styles for dynamic generation:

* Sunrise gradient
* Coffee morning vibe
* Nature landscape
* Soft abstract colors

Each template should support dynamic text overlay.

---

INTERACTION BEHAVIOR:

* When user clicks "Gerar imagem":
  → show loading state
  → display generated image

* WhatsApp button:
  → prepares sharing action (image + text)

---

COMPONENTS TO INCLUDE:

* Reusable button component (large, rounded)
* Input field (simple and clear)
* Card container
* Image preview component
* Loading state (spinner or skeleton)

---

EXPORT REQUIREMENTS:

* Generate clean, developer-friendly layout
* Use auto layout
* Use components
* Prepare for code export (HTML/CSS/React)

---

EXTRA:

Keep everything extremely simple. This is not for young users. Avoid clutter, small text, or complex UI.

Focus on usability and clarity above aesthetics.
